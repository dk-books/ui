# Spacing

Designing within a defined spacing system allows us to work faster and with greater consistency.

Consistent and scaleable spacing also allows us to establish a logical heirarchy across our digital estate.

## Base Unit

Our spacing and grid are based on an 8px base unit.

A 4px half unit is available for fine tuning of small elements and very small text, but should only be used where 8px wouldn't be appropriate.
